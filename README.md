# Voithru AI Translation Factory (Flow)

이 프로젝트는 CrewAI Flow를 사용하여 번역 프로젝트의 인테이크부터 견적 발송까지를 자동화합니다.

## 핵심 흐름
1. **Gmail 검색**: '번역 요청' 관련 이메일 필터링.
2. **데이터 분석**: 본문 및 첨부파일 유효성 검사 (예외 처리).
3. **견적 산출**: 난이도 및 분량 기반 동적 가격 책정.
4. **Slack 알림**: 담당자에게 고액 건 또는 특이사항 보고.
5. **Human-in-the-loop**: 담당자 승인 시 클라이언트에게 최종 견적 발송.
